# Website Maintainer Handbook
**Gamma Iota Lambda Chapter · Alpha Phi Alpha Fraternity, Inc.**

A standalone reference guide for maintaining and updating the official GIL chapter website at [gil1906.org](https://www.gil1906.org). Written for incoming Associate Editors, future webmasters, and anyone handed responsibility for the site who may not have a web development background.

---

## What's in this repo

A single self-contained HTML file — `index.html` — that renders as a fully styled, navigable documentation site. No build tools, no dependencies, no server required. Open it in any browser.

---

| Section | Topic |

| 15 | Annual update checklist |
| 16 | Troubleshooting — common problems and fixes |
| 17 | Learning resources — HTML/CSS, JavaScript, Git, command line, VS Code |

## Viewing the handbook

**Locally:** Clone this repo and open `index.html` in any browser.

```bash
git clone https://github.com/Gamma-Iota-Lambda/website-maintainer-handbook.git
cd website-maintainer-handbook
open index.html        # Mac
start index.html       # Windows
```

**On GitHub Pages:** If GitHub Pages is enabled for this repo, the handbook is accessible directly at:
`https://gamma-iota-lambda.github.io/website-maintainer-handbook/`

---

## Keeping this up to date

The handbook should be updated whenever the main site undergoes structural changes — new pages, updated workflows, new third-party services, or changes to the publish process. Changes follow the same branch → PR workflow described in the handbook itself.
